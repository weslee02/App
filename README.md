This project develops an Android-based offline mobile health (mHealth) application that uses an Edge AI machine learning model to identify tuberculosis (TB) patients who may be at risk of treatment non-adherence. The application will collect patient information including age, number of missed medication doses, distance from the patient's residence to the healthcare facility, and alcohol consumption status. These variables will be used as inputs to a TensorFlow Lite machine learning model running directly on the mobile device. The model will output a prediction indicating whether the patient is at Low Risk, Moderate Risk, or High Risk of treatment non-adherence. The system is designed to operate offline, allowing adherence risk assessment even in areas with limited internet connectivity. Evaluation will be conducted by measuring the model's classification performance using metrics such as accuracy, precision, recall, F1-score, and confusion matrix during testing. In addition, the Android implementation will be evaluated by measuring on-device inference latency and verifying that predictions can be generated correctly and efficiently on a physical smartphone.
